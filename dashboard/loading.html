Test du Dashboard de Gestion de Contenu

Instructions pour utiliser le dashboard

  1. Assurez-vous d'exécuter ce site sur un serveur web (comme Apache, Nginx ou un serveur de développement PHP).
  2. Le dashboard a besoin de PHP pour fonctionner correctement et sauvegarder les modifications.
  3. Ouvrez le Dashboard pour commencer à éditer le contenu du site.
  4. Dans le dashboard, vous pouvez :
    • Modifier les textes, titres et descriptions
    • Changer les images
    • Ajouter/supprimer des éléments (témoignages, FAQ, etc.)
    • Prévisualiser les changements en temps réel
    • Sauvegarder les modifications pour les appliquer au site réel
Note importante : Pour que la fonctionnalité de sauvegarde fonctionne correctement, le fichier PHP save.php doit avoir les permissions nécessaires pour écrire dans les fichiers du site. Assurez-vous que le serveur web a les droits d'écriture sur le dossier du site.

Problèmes connus et solutions

  • Iframe ne charge pas : Si la prévisualisation ne s'affiche pas, c'est probablement dû à la politique de sécurité de Same Origin. Assurez-vous d'accéder au dashboard via un serveur web et non en ouvrant directement le fichier HTML.
  • Erreurs de sauvegarde : Vérifiez que PHP est bien installé et configuré sur votre serveur et que les permissions d'écriture sont correctement définies.
